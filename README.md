Server-tan - AI
===============

Logic for Server-tan - determines how the bot will respond.

Installation
------------

Add this line to your application's Gemfile:

```ruby
gem 'server_tan-ai'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install server_tan-ai
